“How to use it in bootstrap”

  1. First of all import the required Javascript

    <script src="js/jquery.1.9.1.min.js"></script>
<script src="js/modernizr.custom.49511.js"></script>
<script src="js/bootstrap.min.js"></script>
<script src="js/jquery.onecarousel.min.js"></script>

    Aditional to the bootstrap CSS, we have to include the animate.css at the same time. Pleae note that this animate.css is customized, it's different with the one in the original version. 

    <link href="css/bootstrap.css" rel="stylesheet">
<link href="css/bootstrap-responsive.css" rel="stylesheet">
<link href="css/animate.css" rel="stylesheet">
<link href="css/style.css" rel="stylesheet">
  2. Then you've to add the the slider's content (image, text etc) in the HTML, something looks like:
    <div id="myCarousel" class="carousel slide">
  <!-- Carousel items -->
  <div class="carousel-inner">
    <div class="active item">…</div>
    <div class="item">…</div>
    <div class="item">…</div>
  </div>
  <!-- Carousel nav -->
  <a class="carousel-control left" href="#myCarousel" data-slide="prev">&lsaquo;</a>
  <a class="carousel-control right" href="#myCarousel" data-slide="next">&rsaquo;</a>
</div>
  3. Finally use our jQuery plugin to generate the slider, just use oneCarousel instead of carousel: 
    $(document).ready(function() {
    $('#banner').oneCarousel({
	    	// the global animation
		easeIn: 'rotateIn',
		// delay time of the slideshow
		interval: 5000,
		// pause when user hover
		pause: 'hover'
    });
});
    You can find more infromation from http://twitter.github.com/bootstrap/javascript.html#carousel There is only one more option of the carousel, the easeIn, it's the default ease in animation if you don't specify the animation in the element. And like the original OneByOne slider, we can customize each caption text's animation at same time. Just use the add the data-animate in the HTML tag. For example: 
    <div class="carousel-caption move-up">
	<h1 data-animate="rollIn">OneByOne in Bootstrap</h1>
	<p class="text-info" data-animate="bounceInUp">Your favourite OneByOne Slider now is available in bootstrap as Carousel.<span class="label label-important">WoYa!</span></p>
	<a class="btn btn-large btn-primary" href="#" data-animate="bounceInLeft">Download today</a>
</div>
    The animation of each text is rollIn, bounceInUp, bounceInLeft here.
