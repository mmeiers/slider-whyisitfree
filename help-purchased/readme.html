A) FAQ - top

  1. First of all import the required Javascript

    <script src="js/jquery.1.9.1.min.js" type="text/javascript"></script>
<script src="js/modernizr.custom.49511.js" type="text/javascript"></script>
<script src="js/jquery.onebyone.min.js" type="text/javascript"></script>
<script src="js/jquery.touchwipe.min.js" type="text/javascript"></script>

    and the CSS.

    <link rel="stylesheet" type="text/css" href="css/jquery.onebyone.css" />
<link rel="stylesheet" type="text/css" href="css/animate.min.css" />
  2. Then you've to add the the slider's content (image, text etc) in the HTML, which looks like:
    <div id="banner">
	<div class="oneByOne_item">
	  <span class="text1" data-animate="fadeInLeft" >iOS Friendly</span>
	  <span class="text2" data-animate="fadeInRight" >Drag and throw the banner</span>
	  <img src="images/books/book1.png" class="img1" data-animate="rollIn" />
	  <img src="images/books/book2.png" class="img2" />
	</div>
	<div class="oneByOne_item">
		...
	</div>
	<div class="oneByOne_item">
		...
	</div>
	<div class="oneByOne_item">
		...
	</div>
	<div class="oneByOne_item">
	    ...
	</div>
</div>
  3. Finally use our jQuery plugin to generate the slider: 
    $(document).ready(function() {
    $('#banner').oneByOne({
		className: 'oneByOne1',
		easeType: 'random'
	});
});
    The #banner is the slider main container of the slider in the HTML. The oneByOne1 is the generated by the plugin which wrap the #banner. Then customize the css style of each elements in the HTML.
  • The whole available settings of the plugins: 
    $(document).ready(function() {
	$('#banner').oneByOne({
		// the wrapper's name
		className: 'oneByOne',
		// the wrapper div's class name of each slider
		sliderClassName: 'oneByOne_item',
		// pause the auto delay slideshow when user hover
		pauseByHover: true,
 		// the global ease animation style,
		// take effect if you don't pre-defined it in the element
		easeType: 'fadeInLeft',
		// width of the slider
		width: 960,
		// height of the slider
		height: 420,
		// the delay of the touch/drag tween
		delay: 300,
		// the tolerance of the touch/drag
		tolerance: 0.25,
		// enable or disable the drag function by mouse
		enableDrag: true,
		// display the previous/next arrow or not
		showArrow: true,
		// auto hide the arrows when user leave the slider or not
		autoHideButton: true,
		// display the circle buttons or not
		showButton: true,
		// auto play the slider or not
		slideShow: false,
		// the delay millisecond of the slidershow
		slideShowDelay: 3000,
 		// set responsive to true, when you'll change the
		// slider's size with the media query in CSS
		responsive: true
		// when slider is smaller than minWidth,
		// the text are be hidden in the responsive
		minWidth: 480
	});
});
  • How to customize each slider with a different animation? Or even customize each elements in slide with a different animation.
    You can set a global animation when embed the slider in the javascript, you can pre-defined a animation style in each element with data-animate at the same time, the data-animate of the element has a higher priority:
    <img src="images/example1/mouse.jpg"  class="bigImage" data-animate="rollIn"></img>
<span class="text1" data-animate="fadeInLeft">Text Block 1</span>
<span class="text2" data-animate="fadeInRight">Text Block 2</span>
    In the above example, the animation of the image is rollIn, Text block 1's animation is fadeInLeft while Text Block 2's animation is fadeInRight. You can leave the animation class to blank and it will use the global animation.
  • How to make the slider responsive?
    Please note that the responsive only works will the full size image. Which means the image is same size as the slider. And the text block will be hidden in the smaller screen. You can set a minWidth, when the screen is smaller than minWidth, the text blocks will be hidden. When embed in the javascript, set the responsive to true, and define the minWidth: 
    $(document).ready(function() {
       $('#banner').oneByOne({
		className: 'oneByOne1',
		easeType: 'random',
		responsive: true,
		minWidth: 480
	});
   });
    Then in your media query of CSS, change the slider item and it's container's width as you need, something like:
    @media screen and (min-width: 768px) and (max-width: 959px) {
	.oneByOne1, #banner .oneByOne_item{ width: 748px; }
}
  • How many ease animation style does this plugin support?
    We use CSS3(animate.css) to toggle the animation here, so the easeType can be 'fadeIn', 'fadeInUp', 'fadeInDown', 'fadeInLeft', 'fadeInRight', 'fadeInRight', 'bounceIn', 'bounceInDown', 'bounceInUp', 'bounceInLeft', 'bounceInRight', 'rotateIn', 'rotateInDownLeft', 'rotateInDownRight', 'rotateInUpLeft', 'rotateInUpRight', 'fadeInLeftBig', 'fadeInRightBig', 'fadeInUpBig', 'fadeInDownBig', 'flipInX', 'flipInY', 'lightSpeedIn' or 'random'.
    The animate.css is a little big now. You can use the easeType only css to minify the size. Take the example2.html for example. We use the fadeInLeft only. So we can customize the fadeInLeft CSS only in the example2.css. You can find some more information from: http://daneden.me/animate/, and there is a custom build tool there.

B) The source and deploy Folder - top

You only have to upload the files in the deploy folder. The deploy folder contains the minified javascript. You can just update the image content in the index.html by your own ones.

Please don't upload the files in the source folder. The source folder contains the original javascript source, so please don't upload the javascript files in this folder. Thanks.
